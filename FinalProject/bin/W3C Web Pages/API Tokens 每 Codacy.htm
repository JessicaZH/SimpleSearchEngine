API Tokens

Codacy Team
December 13, 2017 13:08
Follow

API Tokens allow you to use the Codacy API.

To generate one, go to your Account Settings and click on the tab API Tokens. 

For more information about the API check the API Docs.

To send coverage for your project, you need the Project API token.
