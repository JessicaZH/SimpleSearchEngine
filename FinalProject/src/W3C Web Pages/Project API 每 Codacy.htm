Project API

Codacy Team
November 08, 2018 11:53
Follow

The Codacy API offers specific methods to manipulate projects. You access these by using a Project API Token generated for your project.

 

Creating a Project API integration in your project Settings tab will generate an API Token valid for that project.

To revoke a Token, remove the integration.
